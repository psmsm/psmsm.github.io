# psmsm.github.io

Winston's Tech Blog - 알고리즘, 강화학습, 엔지니어링 연구 블로그

## 로컬 실행 방법

```bash
# 의존성 설치
bundle install

# 로컬 서버 실행
bundle exec jekyll serve

# 브라우저에서 http://localhost:4000 접속
```

## 글 작성 방법

`_posts/algorithms/` 디렉토리에 `YYYY-MM-DD-title.md` 형식으로 파일 생성

### Front Matter 예시

```yaml
---
title: "[알고리즘] 제목"
excerpt: "요약"
categories:
  - algorithms
tags:
  - 태그1
  - 태그2
---
```

### LaTeX 수식 사용

- 인라인: `$E = mc^2$`
- 블록: `$$\int_0^1 x^2 dx$$`
